# 027 Relationships

[![027 Relationships](https://img.youtube.com/vi/YgRrn7BgR0I/0.jpg)](https://www.youtube.com/watch?v=YgRrn7BgR0I)

The SQL in the `exercise.sql` file
- Should return the following columns `ID`, `NAME`, `EMAIL`, `PHONE_NUMBER`, `LOCATION`, `REVENUE` (some of these columns may exist in multiple table)
- Should use a total of two `INNER JOIN`s on the `Employee`, `Contact_Info` and `Stores` table, using an Employee's `LOCATION` and `ID` for the joins
- Should only return employee's located in Portland, Seattle, Los Angeles and San Francisco
